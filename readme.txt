Очередность флагов в грепе
 -v инвертирует смысл всегда?
Эти флаги меняют вывод полностью - выводят только эту информацию
 -l выводящий только совпадающие файлы
 -с выводит только количество сопадающих строк

 Если есть -v то 
 -nhso ничего не выводят

 вроде как эти флаги равнозначны и друг друга уже не заменяют - работают по отдельности
 -n 
 	Предваряет каждую строку вывода номером строки из файла ввода.
 -h
 	Выводит совпадающие строки, не предваряя их именами файлов.
 -s
 	Подавляет сообщения об ошибках о несуществующих или нечитаемых файлах.
 -o
 	Вроде самый сложный - необходимо выводить по совпадению
